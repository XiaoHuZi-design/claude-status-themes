---
name: sb
description: 快速切换 Claude Code 状态栏主题（仙侠古风/极客仪表盘）
---

当用户执行 `/sb` 命令时，执行以下操作：

## 主题列表

| 命令 | 主题 |
|------|------|
| `geek`, `g`, `极客` | 极客仪表盘 |
| `xianxia`, `x`, `仙侠` | 仙侠古风 |

## 操作步骤

1. 读取用户指定的主题（默认为 `geek`）
2. 执行切换脚本：`bash ~/.claude/switch-theme.sh [主题]`
3. 确认切换成功

## 示例对话

- 用户: `/sb geek` → 切换到极客仪表盘
- 用户: `/sb xianxia` → 切换到仙侠古风
- 用户: `/sb g` → 切换到极客仪表盘（简写）
- 用户: `/sb` → 默认切换到极客仪表盘
